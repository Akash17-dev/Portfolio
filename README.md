# Akash Aakula Portfolio

Premium personal portfolio built with Next.js 15, React 19, TypeScript, Tailwind CSS, Framer Motion, GSAP, Three.js, React Three Fiber, and Lucide Icons.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Experience

- Liquid glass and glassmorphism UI
- Animated aurora background and mouse-follow glow
- 3D React Three Fiber hero object
- Magnetic buttons, tilt cards, animated counters, and GSAP scroll reveals
- Responsive sections for About, Skills, Projects, GitHub, Experience, Ventures, and Contact
